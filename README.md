# Shadow of the Weird Wizard for homebrewery

This is a custom theme for [The Homebrewery](https://homebrewery.naturalcrit.com/) based on the Shadow of the Weird Wizard. You can see a live demo of the theme [here](https://homebrewery.naturalcrit.com/share/4NPg1XCkaf5Q/).

## Getting Started

At the top of your brew simply paste the following line:

```html
<link rel="stylesheet" type="text/css" href = "https://patrickporto.github.io/homebrewery-theme-sofww/sofww.css" />
```
